新媒体必备工具

Single
2018-09-24
全部文章, 学习资源, 实用工具, 干货福利

这世界上有很多事情不是要看到希望才去坚持的、而是要你先坚持了才会看到希望的、小伙伴们加油吧、

我没上传图片也没有详细介绍、很多网页只是随便描述了一下、但是不管你自由职业、自媒体人、不管你从事什么工作、以下分享的这些工具多数是你用得着的、有需要的花上几分钟时间自己看看、不好用的垃圾我是不会写的

一、工具篇

注明

凡是 ： 号前的名字均是超链接

数据指数

  1. 百度指数：很好用的内容选题工具。
  2. 5118：中国目前最好用的网站数据工具。
  3. 新榜：专注公众号数据。
  4. 清博指数：专注公众号数据。
  5. 艾瑞指数：可以查看APP、网站对应的数据情况，方便做渠道投放的评估和竞品研究。

原型设计

  1. 墨刀：国产原型设计工具，支持协同，功能丰富。
  2. Balsamiq Mockup：操作简单，有在线版和客户端版，非常适合新手。
  3. invisionapp：在线的原型交互设计工具，页面美观，使用网页和APP原型，操作简单比 Axure 简单。
  4. Axure PR：老牌原型设计软件，适合专业人士，可做模拟交互。

着陆页制作

  1. 易企秀：同Maka差不多，但是设计感稍微差一些。
  2. ih5：目前中国最强大的H5制作工具，上手具有一定难度。
  3. Strikingly：硅谷的中国团队出品，目前中国最容易上手制作着陆页的工具。

图片设计

  1. 创客贴：国内目前在线设计工具的老大。
  2. Fortor：全球1.9亿用户的在线快速平面设计平台。
  3. 瓦斯：腾讯模仿Canvas做的产品，更适合制作信息图。
  4. Powerpoint：之前自己也用过挺喜欢的。
  5. Sketch：仅限Mac，正在逐步替代PS成为主流的UI设计工具。

图文排版

  1. Markdown Here：公众号最骇客的排版辅助工具。
  2. 秀米：目前使用量最高的公众号排版工具。
  3. i排版：同上，但是页面比较乱。
  4. 135编辑器：同上，但是页面比较乱。

视频制作

  1. Spark：Adobe出品的在线视频制作工具，需要翻墙。
  2. VUE：快速制作电影级的短视频。
  3. 小影：新媒体短视频创作工具。
  4. Powtoon：非常强大的在线动画视频制作工具，就像动画版的PPT工具。
  5. 爱美刻：在线视频制作工具，基于模版，适合做生活小短片。

屏幕录制

  1. Camtasia：Windows 平台最佳录屏工具
  2. Snagit：同为 Camtasia 出品，MAC平台录屏工具，操作简便。

在线预约

  1. https://calendly.com/：在线预约系统的Saas工具，可以自动做预约人数的扣除，邮件通知，缺点是全英文，速度一般

配色工具

1.Kuler ：Adobe出品的强大配色工具。

  1. 配色表：国内网页设计常用的色彩搭配方案。
  2. 千图配色：配色的小工具比较多

公众号辅助

  1. 新媒体管家：内嵌在微信公众号内的运营助手，具有排版、找图、永久链接等辅助功能
  2. 壹伴：和新媒体管家功能类似，浏览器插件很好用、我一直用的就是它、小而美的公司，用户体量稍微小一些。
  3. 微小宝：把同一篇文章同时发布到不同的平台，类似国外的 Buffer。
  4. 西瓜公众号助手：同为辅助工具，比较亮眼的功能包括违规检测、粉丝预估、定时发送等功能。

在线表单

  1. 麦客：个人最喜欢的CRM工具。
  2. 金数据：在某些数据管理上功能更强，但是CRM功能弱。
  3. Typefom：国外优雅至极的表单工具，特别适合做测试题目，用户画像的调研。

活动发布

  1. 活动行：目前国内主流的活动发布工具。
  2. 孤鹿：社群运营工具，也可以方便的发布活动，进行报名收款。

内容电商

  1. 有赞：国内最强大的移动电商工具。
  2. 小鹅通：主要用户课程电商产品。
  3. 千聊：语音直播类课程的工具。
  4. 荔枝微课：和千聊差不多，课程裂变分享的功能做的好一些。

社群变现

  1. 知识星球：微信社群工具，帮助公众号连接粉丝，通过提问、文章向粉丝收费。
  2. 小社群：小鹅通模仿小密圈出品的，如果和课程一块做，可以和小鹅通整合的深一些。

邮件营销

  1. Mailchimp：最强大的邮件营销工具，但是表单验证需要用户翻墙。
  2. 麦客邮件：国内邮件营销最优秀的工具，整合表单CRM。

群机器人

  1. 微友助手：具有新加好友自动应答，自动拉人入群，入群欢迎语，智能聊天，消息群发，自动回复，定时提醒，群签到，群精华，群成员、群文件管理等强大功能。

二维码工具

  1. 草料二维码生成器：功能非常强大的二维码工具，活码、静态码一一俱全。
  2. 第九工厂：可以设计比较个性化的二维码。

短网址

  1. 新浪微博短网址生成器：新浪出品，服务稳定。
  2. suo：缩我(Suo.im)免费专业的网址缩短服务,在线生成短网址,开放API接口,可批量生成短链接,不限制接口请求数,跳转快,稳定可靠,防屏蔽,防拦截!免费提供访问统计!！

图床工具

  1. 极简图床：直接拖拽生成图片外链。

二、素材篇

图标资源库

  1. iconfont：阿里出品的图标库。
  2. TheNounProject：国外最全的免费图标库，种类丰富。

图片资源库

  1. Unslpash：大量高清免费商用图片。
  2. Gratisography：高清风格化摄影图库。

关于免费素材的之前还特意写过一篇文章、很多很多、自己翻博客文章

视频素材库

  1. Mazwai：这是我最喜欢的视频库了，主要集中在风景、建筑主题上，虽然没有严格的分类，但是每一段高清素材都是电影级别的质感。
  2. Pexels：Pexels 拥有大量的免版权的高清素材库（CC0协议｜不用署名也不用付费），按列别主要分为：动物、街道、山川、延时、模糊、食物、城市、饮料、天空等等。你可以把其中的视频放到官方网站、销售引导页以及任何的商业项目中。虽然东西都是免费，但是质量都是极高的。你可以通过邮件实时订阅他们的素材更新。
  3. Wedistill：技术服务公司Create The Bridge的子项目，每10天会上架10段视频，全是免费高清的广告素材。
  4. Videovo：庞大的素材库，有超过3000的视频素材，除了高清视频素材还有动画素材，相对其他素材站比较好的一点是，你可以根据素材关键字和热门度进行检索。
  5. Coverr：Covers 通常每周会上架7个新的视频，视频的分类会比Pexels稍微少一点，但是如果是初创的互联网企业，Tech哪个类别的素材要比Pexels好很多。

设计灵感库

  1. Behance：Behance是作品集平台，重在系统地展示，尤其是系列作品。
  2. Dribbble：dribbble则是一个让设计师发布正在创作中的作品的地方，需要会员邀请才能发布作品，相对Behance种类较少，但是风格更前卫。

配乐素材库

  1. Marmoset：最强大的配乐工具，它能根据你的视频配乐所要表达的情绪、能量、长度、风格、是否带人声、等要素，直接筛选后推送对应的配乐。
  2. Musicbed：纯粹的配乐素材库。
  3. Song Freedom：纯粹的配乐素材库。
  4. Free Music Archive：纯粹的配乐素材库。
  5. Audiosocket：纯粹的配乐素材库。
